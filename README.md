HOW TO build on VS 2013 x64
- Download dependencies binaries :
* GLFW http://www.glfw.org/
* GLEW http://glew.sourceforge.net/
* Freetype http://gnuwin32.sourceforge.net/packages/freetype.htm
- Put them in deps/
- Run cmake
